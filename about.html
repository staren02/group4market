About This Project

📌 What is the Chitechma Student Marketplace?

The Chitechma Student Marketplace is a simple web-based platform designed to allow students of Chitechma University Institute, Buea to buy and sell items among themselves within the school community.

This project was created as a school assignment to demonstrate the use of HTML and CSS in building a real-world web application interface. The platform allows students to list products, browse what others are selling, and connect with each other directly on campus.

❗ The Problem We Are Solving

Chitechma students often face challenges when trying to buy or sell used items such as textbooks, electronics, clothing, and other student essentials. Common problems include:

  • No organised place to advertise items for sale
  • Relying on WhatsApp groups which become disorganised
  • Difficulty finding affordable items on campus
  • No way to know which fellow students are selling
  • Items going to waste when they could be reused

The Chitechma Marketplace solves this by providing a dedicated, organised, and easy-to-use platform just for our school community.

🛠️ Technologies Used

This project was built using only basic web technologies as required for the course:

  • HTML5 — Used to structure all pages and forms
  • CSS3 — Used to style the pages with school colours and layout
  • Images — School logo and product placeholder images

Note: No JavaScript, PHP, or database was used. This is a front-end only prototype.

🚀 Future Improvements

In future versions of this project, the following features could be added:

  • Real login authentication using PHP and MySQL
  • A proper database to store product listings
  • Image upload and display from a server
  • An admin dashboard to manage all listings
  • A search and filter feature for products
  • A messaging/chat system between buyers and sellers
  • Mobile-responsive design for smartphones
  • Online payment integration
