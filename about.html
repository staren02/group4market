About This Project

📌 What is the Chitechma Student Marketplace?

The Chitechma Student Marketplace is a web platform that allows students of Chitechma University Institute, Buea to buy and sell items among themselves within the school community.

Students can register with their email, list products with photos, contact sellers directly, and manage everything through a personal dashboard — all without leaving campus.

❗ The Problem We Are Solving

Chitechma students often face challenges when buying or selling used items such as textbooks, electronics, and clothing. Common problems include:

  • No organised place to advertise items for sale on campus
  • Relying on crowded, disorganised WhatsApp groups
  • Difficulty finding affordable items from trusted fellow students
  • Items going to waste when they could be reused

The Chitechma Marketplace solves this with a dedicated, organised, and easy-to-use platform built just for our school community.

🛠️ Technologies Used

HTML5 — Page Structure
CSS3 — Styling & Responsive Design
JavaScript — Interactivity & Logic
localStorage — Data Storage
FileReader API — Image Uploads

Note: This is a front-end prototype using browser localStorage as the data store. A future version would use a PHP + MySQL backend for real persistent storage.
